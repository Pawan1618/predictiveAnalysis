# AQI Prediction Project

This project aims to predict the Air Quality Index (AQI) using machine learning.

## Structure

- `data/`: Contains raw and processed data.
- `notebooks/`: Jupyter notebooks for data cleaning, EDA, and model training.
- `src/`: Source code for data processing and modeling.
- `models/`: Saved models and scalers.
- `app/`: Streamlit application code.
- `report/`: Project reports and presentations.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   streamlit run app/streamlit_app.py
   ```
